# BlockmanGO Changelog Tracker

[![Discord](https://img.shields.io/badge/Discord-Join%20Server-5865F2?logo=discord&logoColor=white)](https://discord.gg/bVgjq6)

## What This Is

```
APK/XAPK → Scanner (private) → JSON snapshots (public) → Diff reports → You
```

This repo contains **only the output data** — structured JSON files produced by scanning every BlockmanGO APK version. The scanning tool itself is private.

## Version Snapshot Format

Each version produces 15+ JSON files. Start with `_index.json` for the summary.

```json
{
  "_meta": {
    "filename": "Blockman+Go_3.21.1.xapk",
    "size_mb": 200.7,
    "md5": "...",
    "sha256": "..."
  },
  "manifest": {
    "package": "com.sandboxol.blockymods",
    "version_name": "3.21.1",
    "version_code": 5671,
    "min_sdk": 23,
    "target_sdk": 35
  },
  "permissions": {
    "total": 45,
    "dangerous": 8,
    "custom": 12
  },
  "activities": { "count": 235 },
  "strings": {
    "total_unique": 436176,
    "urls_count": 311,
    "ips_count": 89
  }
}
```

## Quick Start — Using the Data

### Find all API endpoints added in v3.21.1

```bash
# Machine-readable diff
cat reports/v3.20.3_to_v3.21.1.json | jq '.api_endpoints.added'
```

### Find all activities in the mining system

```bash
cat versions/v3.21.1_build5671/activities.json | jq '.list[] | select(. | contains("mining"))'
```

### Find all Tencent URLs

```bash
cat versions/v3.21.1_build5671/strings.json | jq '.urls[] | select(. | contains("tencent"))'
```

### Compare permissions between versions

```bash
diff <(cat versions/v2.95.3_build5123/permissions.json | jq -S .requested) \
     <(cat versions/v3.21.1_build5671/permissions.json | jq -S .requested)
```

## Running the Scanner

The scanner tool is **private** (gitignored) but documented in [`changelog/README.md`](changelog/README.md).

```bash
# Install dependency
pip install androguard

# Scan a new version (auto-compares, saves snapshot, creates archive)
python changelog/track_versions.py "Blockman+Go_3.21.1.xapk"

# Create baseline (first version)
python changelog/track_versions.py --create-baseline "Blockman+Go_2.95.3.xapk"

# List tracked versions
python changelog/track_versions.py --list

# Quick scan (no save)
python changelog/compare_apk.py app.apk
```

## Security Findings

| Finding | Severity |
|---------|----------|
| No certificate pinning | High |
| MD5 request signing (broken) | High |
| Static API keys (never rotate) | High |
| User IDs as plain integers | Medium |
| Game server IPs leaked | Medium |
| Analytics over HTTP | Medium |
| RSA 1024-bit (below 2048 minimum) | Low |
